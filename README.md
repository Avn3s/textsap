## Installation

Make sure python (version 3.10 or above) and pip are installed first.

Install the dependencies using:
```bash
pip install textual
pip install pygame
```

Then clone this repository using

```git
git clone https://github.com/Avn3s/textsap.git
```

Then run the program
```bash
cd textsap
python sap.py
```
